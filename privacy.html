Política de Privacidad

AFA Web Development desarrolla aplicaciones y juegos móviles. Esta página se utiliza para informar a los usuarios sobre nuestras políticas con respecto a la recopilación, uso y divulgación de información.

Recopilación de información

AFA Web Development no recopila información personal identificable directamente a través de este sitio web.

Sin embargo, nuestras aplicaciones pueden utilizar servicios de terceros que pueden recopilar información utilizada para identificar a los usuarios.

Servicios de terceros

Las aplicaciones desarrolladas por AFA Web Development pueden utilizar servicios de terceros, incluyendo pero no limitándose a:

  • Google Play Services
  • Google AdMob

Estos servicios pueden recopilar información como identificadores de dispositivo, datos de uso y diagnósticos, de acuerdo con sus propias políticas de privacidad.

Publicidad

AFA Web Development utiliza Google AdMob para mostrar anuncios. AdMob puede usar identificadores publicitarios del dispositivo para mostrar anuncios relevantes a los usuarios.

Seguridad

Valoramos la confianza de los usuarios y nos esforzamos por utilizar medios comercialmente aceptables para proteger la información. No obstante, ningún método de transmisión por Internet o almacenamiento electrónico es 100% seguro.

Enlaces a otros sitios

Este sitio puede contener enlaces a otros sitios web. No somos responsables del contenido ni de las prácticas de privacidad de dichos sitios.

Cambios a esta política

Esta Política de Privacidad puede actualizarse ocasionalmente. Se recomienda revisar esta página periódicamente para detectar cambios.
